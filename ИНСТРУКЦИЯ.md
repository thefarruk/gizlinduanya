# 🚀 ИНСТРУКЦИЯ ЗАПУСКА — GizlinDünya

## ШАГ 1 — Supabase (база данных, бесплатно)

1. Зайдите на https://supabase.com
2. Нажмите "Start your project" → войдите через GitHub
3. Создайте новый проект:
   - Name: gizlinduanya
   - Password: придумайте надёжный пароль
   - Region: выберите ближайший (EU West)
4. Подождите 2 минуты пока проект запустится
5. Зайдите в раздел "SQL Editor"
6. Вставьте содержимое файла supabase_schema.sql
7. Нажмите "Run" — все таблицы создадутся автоматически
8. Зайдите в Settings → API
9. Скопируйте:
   - Project URL (это NEXT_PUBLIC_SUPABASE_URL)
   - anon public key (это NEXT_PUBLIC_SUPABASE_ANON_KEY)

---

## ШАГ 2 — GitHub (хранение кода, бесплатно)

1. Зайдите на https://github.com
2. Создайте новый репозиторий: gizlinduanya (private)
3. Установите Git на компьютер если не установлен
4. В папке проекта выполните:

```bash
git init
git add .
git commit -m "первый запуск"
git remote add origin https://github.com/ваш-ник/gizlinduanya.git
git push -u origin main
```

---

## ШАГ 3 — Vercel (хостинг, бесплатно)

1. Зайдите на https://vercel.com
2. Войдите через GitHub
3. Нажмите "New Project"
4. Выберите репозиторий gizlinduanya
5. В разделе "Environment Variables" добавьте:

```
NEXT_PUBLIC_SUPABASE_URL = ваш URL из Supabase
NEXT_PUBLIC_SUPABASE_ANON_KEY = ваш ключ из Supabase
JWT_SECRET = любая длинная строка например: gizlin2025secretkey!@#
```

6. Нажмите "Deploy"
7. Через 2 минуты сайт будет доступен по адресу:
   https://gizlinduanya.vercel.app

---

## ШАГ 4 — Свой домен (опционально)

Купить домен можно на:
- namecheap.com (от $8/год)
- godaddy.com (от $10/год)

В Vercel → Settings → Domains → добавить свой домен.
